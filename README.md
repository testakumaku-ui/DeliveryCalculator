## 📌 Описание

Проект для расчёта стоимости доставки с учётом:

- типа доставки
- веса
- расстояния
- срочности
- страховки
- промокодов

Проект выполнен в рамках задания по рефакторингу legacy-кода.

## 🛠 Что было сделано

- Выполнен рефакторинг исходного кода
- Удалено дублирование логики
- Заменены строковые значения на enum
- Добавлены модели данных (Request/Result)
- Реализована валидация входных данных
- Код разбит на логические методы
- Добавлены unit-тесты (xUnit)

## 🧮 Бизнес-логика

### Тип доставки
- Standard — 200
- Express — 500
- Cargo — 1000

### Вес
- до 1 кг — 0
- до 5 кг — 100
- до 20 кг — 300
- более 20 кг — 800

### Расстояние
- до 10 км — 100
- до 50 км — 300
- до 200 км — 800
- более 200 км — 1500

### Дополнительно
- Срочность — зависит от типа
- Страховка — зависит от типа

### Промокоды
- SALE5 — скидка 5%
- SAVE300 — скидка 300

## 🧪 Тестирование

Используется xUnit.

Реализованы тесты:

- Валидация входных данных
- Расчёт стоимости
- Проверка скидок
- Проверка граничных значений

Всего: 13 тестов
